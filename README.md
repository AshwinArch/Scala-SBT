This repository contains several examples of core concepts of Scala and SBT (Scala Build Tool). It is a great resource to start learning or to quickly brush up on concepts.

To get started with this project, clone the repository and follow the setup instructions below.

```bash
git clone https://github.com/AshwinArch/Scala-SBT.git
cd Scala-SBT
```

Explore the different examples provided to understand core Scala concepts, especially related to SBT. Each example is self-contained and demonstrates a specific concept.
